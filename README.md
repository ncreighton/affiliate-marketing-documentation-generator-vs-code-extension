# Affiliate Marketing Documentation Generator VS Code Extension

> Auto-generate API docs, README files, and inline comments from Affiliate Marketing code patterns. Saves 2-3 hours per week on documentation.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.3b905ea5`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Affiliate Marketing Documentation Generator VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `3b905ea5.enabled` | `true` | Enable/disable the extension |
| `3b905ea5.autoRun` | `false` | Auto-run on file open |
| `3b905ea5.logLevel` | `"info"` | Log verbosity |

## Requirements

- VS Code 1.74.0 or higher
